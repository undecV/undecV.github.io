推薦的 Windows 應用軟體列表

Posted on Sun 13 December 2020 in Category

推薦一些 Windows 平台下的軟體， 適合日常以及非日常使用。
並不一定每一款軟體你都需要或適合你，也未必是大眾的選擇；
但是那些無時無刻不在彈廣告的辣雞軟件一定不會是你的選擇。

裝機必備的實用軟體大合集！ 論如何優雅的使用你的電腦。

Version 5.0.0 (2020-12-13): 第五版更新。更新和修正了部分內容；程式碼重寫，文章生成基於 Python，jinja2 和 YAML。

概論

提示，本文討論的範圍是 應用軟體，不包括例如 Google Docs, Spotify, Telegram 之類的 網頁應用和服務。

也請注意，這裡的選擇標準非常的... 主觀。（不是排名，不分先後）
以下是粗略的選擇的標準：

  • 確實能夠解決問題，且有效的解決問題；
  • 沒有騷擾性廣告；
  • 界面端莊優雅。
  • FOSS，即是 自由軟體（Free Software） 以及 開放原始碼軟體（Open Source Software，OSS），這裡的 Free 表示「自由（libre）」而不是「免費（gratis）」；開放原始碼軟體保留一部分權利並允許用戶學習、修改、增進提高這款軟體的品質。自由軟體可以不受限制地自由使用、複製、研究、修改和分發。自由軟體更加的開放，而對於開源軟體更加嚴格的定義。

第三版更新

  • 由於篇幅刪除了 瀏覽器插件 章節，瀏覽器的 插件、UserScript、UserStyles 能大幅增益網路使用體驗，以後如果有機會我會再撰文介紹。
  • 刪除了 防毒軟體 章節，免費的防毒軟體體驗確實差（其實付費也是），如果你使用的是正版 Windows 作業系統，自帶原生的 Windows Security 就可以滿足需求。電腦的安全更需要的是建立安全意識，並且保持良好的網路使用習慣，不要看網路上的野路子教學一頓風騷操作說關就關防火牆和防毒軟體之類的。

這份列表從大學的一個突發奇想到現在已四年有餘，感謝各位的捧場，
你們與我的討論也讓我感受到了編寫這份列表的價值。
希望能對各位有所幫助，在完成電腦上繁雜工作的時候可以感到舒適和效率：）

標記

本文的標記可能並不完全或有誤，甚至隨着時間的流逝而與當時不符，所以僅供參考，若發現有漏標或錯標，歡迎提出修改建議：）

  • 沒有標記的，一般來說都是免費軟體（Freeware）。
  • 自由：FOSS，自由軟體（Free Software）和開放原始碼軟體（Open Source Software，OSS）
  • 商業：商業軟體（Commercial Software），一般來說，要錢買的。但有免費版本的不標示。

推薦軟體列表

  • Web Browser - 瀏覽器

    • 自由 Mozilla Firefox: 讓你的心智與網路一樣自由開闊。
    • Google Chrome: 無比簡易、安全又快速的使用體驗。

  • Internet - 網路

    • Download - 下載

      • 自由 aria2: 多協議命令行下載器，功能強大，高學習成本，需調教；但是，用會了就不會再考慮其他下載器了。 (小众软件)
      • 自由 qBittorrent: 自由的 BitTorrent 客戶端，擁有可訂製的種子搜尋功能。 (PortableApps, 知乎)

    • Email - 電子郵件

      • 自由 Mozilla Thunderbird: 與 Firefox 師出同門 Mozilla，老牌、自由的電郵軟體。

    • Remote Access - 遠端

      • 自由 WinSCP: SFTP 客戶端。
      • 自由 PuTTY: SSH 客戶端。
      • Chrome Remote Desktop: Google 出品，免費、快速、簡單的遠端桌面。
      • 商業 TeamViewer: 老牌遠端桌面。

  • Multimedia - 多媒體

    • Audio - 音訊

      • foobar2000: 音樂播放器，專業強大，需要調教。
      • AIMP: 類 Winamp，來自戰鬥民族的高顏值音樂播放器，支持換外觀。
      • MiniLyrics: 為多種播放器自動下載並顯示歌詞。
      • 自由 Audacity: 錄音，音訊剪輯、處理。

    • Video - 視訊

      • 自由 K-Lite Codec Pack: 播放器和編解碼器的軟體合集，專業、設定多、功能強大。
      • 自由 VLC media player: 支持多格式的跨平臺播放器，支持串流，支持換外觀。
      • 小丸工具箱: 簡單易用的壓制工具。
      • 自由 Avidemux: 簡單的影片處理。
      • 自由 Open Broadcaster Software: 傳說中的 OBS，實況、直播串流、錄屏。
      • 自由 MKVToolNix: MKV 打包解包工具。
      • 自由 Aegisub: 字幕時間軸。

    • Image - 圖像

      • 自由 ImageGlass: 輕量級圖片瀏覽器。
      • IrfanView: 快速小巧且功能強大的圖片瀏覽器，不僅僅是看圖。 (善用佳软)
      • GifCam: GIF 錄屏，動圖生成。
      • Snipaste: 截圖 + 貼圖，功能強大，簡單易用。

    • Comic Reader - 漫畫閱讀

      • ComicsViewer: 看漫畫。
      • YACReader: 漫畫閱讀、管理器。
      • 自由 Sumatra PDF: 也可以看漫畫，觸屏友好，見 Office / PDF 章節。
      • 自由 Calibre: 也可以看漫畫，見 Office / EPUB 章節。

    • Creation - 創作

      • 商業 Adobe 全家桶: [Family] 不二的選擇。
      • 自由 Blender: 著名 3D 創作軟體，還能剪影片。
      • 自由 GIMP: 點陣圖編輯軟體，自由軟體界的 Photoshop。
      • 自由 Inkscape: 矢量圖編輯軟體，自由軟體界的 Ai 或 CorelDRAW。
      • 自由 Krita: KDE 家族的繪圖軟體，自由軟體界的 SAI（？）。

  • Embellish and Easy Access - 美化和便捷使用

    • 自由 Rainmeter: 桌面小部件，系統美化的必備。
    • QTTabBar: 為檔案總管添加標籤頁功能。 (小众软件)
    • 自由 Carnac: 在屏幕上顯示鍵盤輸入。
    • 商業 Wallpaper Engine: 整個桌布都是我的播放器，可高度客製化的動態桌布。

  • Files - 檔案處理

    • 自由 Git: 分佈式版本管理。
    • SpaceSniffer: 以「矩形式樹狀結構繪圖法」檢視硬碟佔用空間分佈。
    • 自由 WinMerge: 資料夾比較、同步。
    • 自由 dupeGuru: 尋找重複檔案。

    • Compression - 壓縮

      • 自由 7-Zip: 7z 格式原生，高壓縮率。
      • 自由 PeaZip: 多格式支持。
      • 商業 Bandizip: 多格式支持，方便的文字編碼轉換；v7 新版本免費添加廣告，（可以退回 v6 版本或課金享受。
      • 商業 WinRAR: RAR 格式原生，可添加恢復用冗餘。（其實不要錢也能用）

    • Desktop Search - 桌面搜尋

      • Everything: 超高速文件搜尋。 (小众软件)
      • 自由 Launchy: 快速啓動工具，再也不需要尋找快捷方式。 (小众软件)

  • Security - 安全

    • 自由 Eraser: FBI等級徹底地擦除檔案，幾乎無法被恢復。
    • 自由 VirtualBox: 虛擬機器。

  • Office - 辦公室＼文書處理

    • PDF - 可攜式文件格式

      • Adobe Acrobat Reader DC: 原生 PDF 閱讀器。
      • 自由 Sumatra PDF: 小巧快速的 PDF、電子書、漫畫閱讀器，觸屏友好。

    • Database - 資料庫

      • 商業 Memento Database: 強大易用的資料庫管理應用，配合 Android 版食用風味更加。

    • Office suite - 辦公室套件

      • 自由 LibreOffice: 自由軟體界的 M$ Office。
      • 自由 OpenOffice.org: 自由軟體界的 M$ Office。

    • Mind Map＼心智圖＼思維導圖

      • 自由 FreeMind: 心智圖繪製軟體。

    • EPUB

      • 自由 Calibre: 電子書書櫃、EPUB、漫畫閱讀器。
      • 自由 Sigil: EPUB 編輯器。

    • Text editor - 文本编辑器

      • 自由 Notepad++: 純文字＼程式碼編輯器，代替記事本，從此再也不怕亂碼。
      • 自由 Visual Studio Code: Microsoft 出品，Visual Studio 系列，純文字＼程式碼編輯器，強大的擴展性，假裝是編輯器的 IDE。

  • Languages - 語言

    • 自由 RIME: 擊響中文之韻。中文輸入法引擎，支持注音、拼音、五筆等各種常見和不常見的輸入法，專業強大、跨平臺、自由度極高、可高度定製、校難上手。
    • 自由 Locale-Emulator: 轉區工具。

  • Application Utilities - 應用工具

    • 自由 BitMeter OS: 網路流量監視器。
    • 自由 Pomodoro: 簡潔、可換外觀的番茄工作法計時器。
    • 自由 EarTrumpet: 給軟體指定聲音設備、控制音量，需要 Win10 1803 以上。
    • 自由 AutoHotkey: 強大的自動化工具，需要一點寫手稿語言能力。
    • 自由 Microsoft PowerToys: Windows 官方出品，更靈活的視窗佈局、圖片改大小、批量改檔名等實用功能。
    • Microsoft Garage Mouse without Borders: 無縫在多台的電腦上分享切換一套鍵鼠。
    • Moo0 家族: [Family] 45 款小巧實用的工具合集，好看易用。
    • xiles 家族: [Family] 簡單、精心設計和易於使用的軟體。
      • nexusfile: 檔案總管。
      • nexusfont: 字形瀏覽器。
      • nexusimage: 圖片瀏覽器。

  • System Utilities - 系統工具

    • 自由 Crystal Disk: [Family] 可愛的水晶雫幫你監測硬碟狀態。
      • 自由 CrystalDiskInfo: 硬碟狀態監測。
      • 自由 CrystalDiskMark: 硬碟讀寫測速。

  • Package Manager - 程式管家

    • Steam: 數位發行平台，遊戲、軟體、影片、音樂...
    • 自由 PortableApps.com: 便攜軟體，使得軟體不依賴電腦。
    • 自由 Chocolatey: 軟體包管理器，Windows 的 apt 或 yum。

如何尋找實用優秀軟體

怎麼找？

  • 電腦的軟體的強大，不在於能做到多少事情，而是一件事情能做得多好。
  • 應用軟體應該旨在幫助用戶完成工作，提高效率。 而不是讓用戶的電腦成為廣告投放機。
  • 遠離流氓軟件，享受健康生活。
  • 一些很棒的軟體功能非常強大，但是學習門檻較高，或是需要經過調教，甚至可能需要老司機帶路，但是學會了就能夠享受它們的便利了。
  • 不要小看網頁瀏覽器的附加元件＼擴充套件，他們能夠極大地提高體驗和便利。
