List of Windows Application Software

Posted on Mon 15 April 2019 in Category

推薦一些 Windows 平台下的軟體， 適合日常以及非日常使用。 並不一定每一款軟體你都需要或適合你，也未必是大眾的選擇； 但是那些無時無刻不在彈廣告的辣雞軟件一定不會是你的選擇。

裝機必備的實用軟體大合集！ 論如何優雅的使用你的電腦。

概論

請注意，這裡的選擇標準非常的... 主觀。（不是排名，不分先後） 以下是粗略的選擇的標準，但是不一定全都滿足：

  • 確實能夠解決問題，且有效的解決問題；
  • 沒有騷擾性廣告；
  • 開放原始碼軟體（Open Source Software，OSS），保留一部分權利並允許用戶學習、修改、增進提高這款軟體的品質。
  • 自由軟體（Free Software），這裡的 Free 表示「自由（libre）」而不是「免費（gratis）」；自由軟體可以不受限制地自由使用、複製、研究、修改和分發。自由軟體都是開源軟體，而自由軟體更加的開放，而對於開源軟體更加嚴格的定義。
  • 界面端莊優雅。

標記

本文的標記並不完全，僅供參考，若發現可能有漏標或錯標，歡迎提出修改建議：）

  • 沒有標記的，一般來說都是免費軟體（Freeware）。
  • 開源：開放原始碼軟體（Open Source Software，OSS）
  • 自由：自由軟體（Free Software）
  • 商業：商業軟體（Commercial Software），一般來說，要錢買的。但有免費版本的不在標示範疇。

推薦軟體列表

Web Browser with add-ons - 網頁瀏覽器以及附加元件

  • Firefox: 自由 讓你的心智與網路一樣自由開闊。
    • Extensions
      • Basic - 基礎
        • Adblock Plus: 簡稱 ABP，著名廣告攔截插件。
        • Copy as Markdown: 以 Markdown 等格式复制 URL，標題。
        • Copy URL To Clipboard: 复制 URL，標題到剪贴板，便於分享等。
        • Country Flags & IP Whois: 顯示網站所在國家的國旗。
        • Firefox Multi-Account Containers: Firefox 官方出品，小號多開，帳號隔離。
        • Flagfox: 顯示網站所在國家的國旗。
        • OneTab: 將當前所有的分頁收起來並生成一個列表，加快瀏覽器速度。
        • Pushbullet: 快速在各種裝置之間傳遞訊息和檔案。
        • Tab Center Redux: 在側邊欄管理分頁（Tabs）。
        • Undo Close Tab: 一鍵重新打開最近關閉的分頁按鈕。
        • Firefox Color: 客製化你的 Firefox 外觀。
        • Markdown Here: 使用 Markdown 撰寫 EMail，可以與 Evernote 搭配使用。
        • Popup window: 使用迷你視窗打開網頁。
        • YouTube Popout Player: 使用迷你視窗打開 Youtube 影片。
      • Privacy - 隱私保護
        • Facebook Container: 防止 Facebook 窺探和竊取你的隱私。
        • Google search link fix: 修正 Google 等搜尋引擎結果的重定向，防止搜尋引擎記錄你的點擊。
        • HTTPS Everywhere: 電子前哨基金會 EFF 出品，加密的你的通訊，在可以的情況下儘量使用 HTTPS 訪問網頁，更加的安全。
        • Privacy Badger: 電子前哨基金會 EFF 出品，屏蔽間諜廣告和追蹤腳本。
      • Special - 特殊需求
        • FoxyProxy Standard: 管理切換代理伺服器，可用於科學上網。
        • Steam Database: 顯示 Steam 商品的詳細信息，例如價格變化。
        • Unblock Youku: 在海外看中國視頻網站，解除地區限制。
        • 购物党实时比价工具: 中國網路商城比價。
        • 云盘万能钥匙: 自動填寫百度云盤密碼，檢測鏈接是否有效。
        • Aria2 Download Manager Integration: 與 Aria2 下載器搭配使用。
      • Advanced - 進階
        • Greasemonkey: User Script 腳本必備。
        • Stylus: User Styles 必備，可以客製化修改一些網站（例如 Google）的外觀樣式。
        • New Tab Override: 自己定義新分頁顯示的內容。
  • Google Chrome: 瀏覽速度飛快。不僅快速簡單好上手，而且安全無虞，適用各種裝置。

如何尋找實用優秀軟體

怎麼找？

  • 電腦的軟體的強大，不在於能做到多少事情，而是一件事情能做得多好。
  • 應用軟體應該旨在幫助用戶完成工作，提高效率。 而不是讓用戶的電腦成為廣告投放機。
  • 遠離流氓軟件，享受健康生活。
  • 一些很棒的軟體功能非常強大，但是學習門檻較高，或是需要經過調教，甚至可能需要老司機帶路，但是學會了就能夠享受它們的便利了。
  • 不要小看網頁瀏覽器的附加元件＼擴充套件，他們能夠極大地提高體驗和便利。
