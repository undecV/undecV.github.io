概論

提示，本文討論的範圍是 瀏覽器擴充套件； 並且不包括太包括一些 SaaS 網頁應用和服務所附帶的擴充套件。

同樣地，文章其中的一些擴充套件，我可能不常用，甚至時下可能還沒有安裝，但我一定都有用過、體驗過。

也請注意，這裡的選擇標準非常的... 主觀。（不是排名，不分先後）
以下是粗略的選擇的標準：

  • 確實能夠解決問題，且有效的解決問題；
  • 沒有騷擾性質的廣告；
  • 界面端莊優雅，易於使用。
  • FOSS，即是 自由軟體（Free Software） 以及 開放原始碼軟體（Open Source Software，OSS），這裡的 Free 表示「自由（libre）」而不是「免費（gratis）」；開放原始碼軟體保留一部分權利並允許用戶學習、修改、增進提高這款軟體的品質。自由軟體更加的開放，可以不受限制地自由使用、複製、研究、修改和分發。

標記

本文的標記可能並不完全或有誤，甚至隨着時間的流逝而與現時不符，所以僅供參考，若發現有漏標或錯標，又或是你有更好的軟體選擇，歡迎在這個 Repo 提出修改建議：）

許可協議：

  • 沒有標記的，一般來說都是免費軟體（Freeware）。
  • 自由：FOSS，自由軟體（Free Software）和開放原始碼軟體（Open Source Software，OSS）
  • 商業：商業軟體（Commercial Software），一般來說，要錢買的。但有免費版本的不標示。

以及標籤：

  • By Firefox：Mozilla Firefox 官方出品
  • AMO Recommended：Firefox 擴充套件站推薦
  • Tor Included：Tor 瀏覽器內置

推薦軟體列表

By Firefox - Mozilla Firefox 官方出品

  • 自由 Firefox Color: (Add-on) (Home Page) (GitHub) 快速客製化 Firefox 瀏覽器的外觀。 [# By Firefox]
  • 自由 Firefox Multi-Account Containers: (Add-on) (GitHub) 多重賬戶登錄（Cookies 隔離）。 [# By Firefox]
  • 自由 Facebook Container: (Add-on) (GitHub) 隔離 Facebook，阻擋 Facebook 的追蹤器以保護隱私。 [# By Firefox]

Android - 行動端

  • 自由 Google Search Fixer: (Android Add-on) (GitHub) 改善 Google 搜尋在 Firefox Android 上的體驗。

Tabs - 分頁增強

  • 自由 Undo Close Tab: (Add-on) (GitHub) 再次打開最近關閉的分頁。 [# AMO Recommended]
  • Tree Style Tab: (Add-on) (Home Page) (GitHub) 以樹狀的形式列出開啟的分頁。 [# AMO Recommended]
  • 自由 FoxyTab: (Add-on) 增加超多分頁相關便利的操作。 [# AMO Recommended]
  • 自由 SingleFile: (Add-on) (GitHub) 將網頁存檔為單一個可完全離線閱讀的 HTML 檔。 [# AMO Recommended]
  • 自由 Popup window: (Add-on) (GitHub) 用沒有工具列的簡潔的彈出式視窗打開分頁。
  • OneTab: (Add-on) (Home Page) 快速按組收納分頁到列表。 [# AMO Recommended]
  • 自由 Tabliss: (Add-on) (Home Page) (GitHub) 好看、可客製化的新分頁。 [# AMO Recommended]

Ad Blocker & Privacy - 擋廣告和隱私保護

  • 自由 HTTPS Everywhere: (Add-on) (Android Add-on) (Home Page) (GitHub) 電子前哨基金會出品，優先使用更安全的 HTTPS 協定代替不安全的 HTTP 瀏覽網頁。 [# AMO Recommended, # EFF, # Tor Included]
  • 自由 Privacy Badger: (Add-on) (Android Add-on) (Home Page) 電子前哨基金會出品，阻擋追蹤器。 [# AMO Recommended, # EFF]
  • 自由 uBlock Origin: (Add-on) (Android Add-on) (Wikipedia) (GitHub) 擋廣告。 [# AMO Recommended]

Google Enhancement - Google 增強

  • 自由 uBlacklist: (Add-on) (Home Page) 封鎖指定搜尋結果。
  • 自由 To Google Translate: (Add-on) (GitHub) 快速使用 Google 翻譯。 [# AMO Recommended]
  • 自由 Google search link fix: (Add-on) (GitHub) 取消 Google 搜尋結果的重新导向。
  • 自由 YouTube Video and Audio Downloader: (Add-on) (Home Page) 下載 YouTube 的影片。 [# AMO Recommended]
  • 自由 YouTube Popout Player: (Add-on) (Home Page) 用沒有工具列的簡潔的彈出式視窗打開 YouTube 影片。
  • 自由 Content Farm Terminator: (Add-on) (GitHub) 標示並封鎖內容農場。

UserScript & UserStyle - 油猴腳本和客製外觀

  • 自由 Greasemonkey: (Add-on) (Home Page) UserScript 油猴腳本。
  • Tampermonkey: (Add-on) (GitHub) UserScript 油猴腳本。
  • 自由 Stylus: (Add-on) (GitHub) UserStyle 客製外觀。 [# AMO Recommended]

Theme & Style - 主題和外觀

  • 自由 Dark Reader: (Add-on) (Android Add-on) (Home Page) (GitHub) 為每個網頁開啟夜間＼黑暗模式。 [# AMO Recommended]

Web Service - 服務

  • 自由 Steam Database: (Add-on) (Home Page) (GitHub) Steam 歷史價格個其他資訊。
  • Pushbullet: (Add-on) (Home Page) 傳送分頁、訊息、或任何東西到手機或是其他設備。
  • 自由 Privacy Pass: (Add-on) (Home Page) 快速通過 Cloudflare 和 hCaptcha 的人機驗證。
  • Octotree: (Add-on) (Home Page) 以樹的方式展示 GitHub 的 Repo 的檔案。

Copy and Paste - 拷貝和粘貼

  • 自由 Copy URL To Clipboard: (Add-on) (GitHub) 以各種格式拷貝鏈接。
  • 自由 Copy as Markdown: (Add-on) (GitHub) 以 Markdown 格式拷貝鏈接、分頁、圖片。
  • 自由 Copy PlainText: (Add-on) 以無格式拷貝文字。
  • 自由 Absolute Enable Right Click & Copy: (Add-on) (GreasyFork) 強制啟用滑鼠右鍵選單和拷貝功能。

Browser Configuration - 瀏覽器設定

  • 自由 Bookmark Dupes: (Add-on) (GitHub) 尋找和清理重複書籤。
  • 自由 Cookie Quick Manager: (Add-on) (GitHub) 編輯 Cookies。 [# AMO Recommended]
  • 自由 FoxyProxy Standard: (Add-on) (Home Page) (GitHub) 功能強大的代理設定。 [# AMO Recommended]
  • 自由 Quick Accept-Language Switcher: (Add-on) (GitHub) 快速切換網頁請求中 Accept-Language。
  • 自由 User-Agent Switcher: (Add-on) (Home Page) 快速切換網頁請求中 User-Agent。

Utilities - 實用工具

  • 自由 Markdown Here: (Add-on) (Home Page) (GitHub) 用 Markdown 編寫郵件。
  • Feedbro: (Add-on) (Home Page) RSS 聚合、閱讀器。 [# AMO Recommended]
  • 自由 Aria2 Download Manager Integration: (Add-on) (GitHub) 用 Aria2 下載檔案。
